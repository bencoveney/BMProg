# ts-react-boilerplate

Includes:
- TypeScript
- React
- Linting
  - tslint
  - tslint-react
- Webpack
  - Development config with dev server
  - Production config with compression
  - HTML Generation
  - Source Maps
  - Favicons
- Reset CSS

Scripts:
- `npm start` - Start developing.
- `npm run build` - Compile once.
